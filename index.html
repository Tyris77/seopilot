Audit checks

8 on-page signals audited in one click

📝

Title Tag

Checks presence, length (30-60 chars), and keyword placement.

🔍

Meta Description

Verifies the description exists and falls within 120-160 characters.

H1

H1 Tag

Ensures exactly one H1 is present and not empty.

🖼

Image Alt Text

Flags images missing alt attributes.

🔗

Links

Counts internal and external links, highlights broken anchors.

🧩

Schema Markup

Detects JSON-LD or microdata structured data.

🔒

HTTPS

Confirms the page is served over a secure HTTPS connection.

📱

Mobile Viewport

Checks for a proper viewport meta tag.
